使用腾讯微云和批处理写文章

Jelon 发表于 2020-03-28

之前使用的是git来保存文章，同时用到 github actions 来实现自动部署，可能是之前一直写程序，CI、CD思想根深蒂固。每次写完博客，提交git，过几分钟后网站生成。

但是发现其实好像脱离写文章本省。

现在使用本地环境写文章，写完之后执行一条 hexo d 命令即可，甚至可以将命令写成脚本，双击一下即可。

至于文件备份，我使用了腾讯微云，会自动将文章更新到云端，保证不丢失
